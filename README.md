# 📊 مشروع تحليل بيانات باستخدام Excel

## 📝 الوصف
هذا المشروع يهدف إلى تحليل بيانات شركة **Ferns and Petals (FNP)**، وهي شركة متخصصة في الهدايا (زهور، كعك، هدايا) وتعمل على مستوى محلي ودولي.  
التحليل يركز على فهم التحديات التجارية مثل رضا العملاء، الإيرادات، مشاكل التسليم، والتخزين الزائد في مواسم معينة.

---

## 📂 هيكل المشروع
- `data/` : ملفات البيانات الخام (CSV, XLSX).
- `excel/` : ملفات Excel التي تحتوي على خطوات التحليل (تنظيف البيانات، الأعمدة المحسوبة، الجداول المحورية).
- `dashboard/` : لوحة التحكم التفاعلية النهائية.
- `results/` : صور أو تقارير مستخرجة من التحليل.
- `README.md` : ملف التوثيق الرئيسي.

---

## ⚙️ الأدوات والتقنيات المستخدمة
- برنامج: Microsoft Excel
- أدوات:
  - **تنظيف البيانات**: تعديل التنسيقات (مثل أرقام الهواتف كنص).
  - **الأعمدة المحسوبة**: إضافة أعمدة مثل الإيرادات.
  - **ربط الجداول**: ربط بيانات العملاء، المنتجات، والطلبات.
  - **Pivot Tables & Charts**: للإجابة على الأسئلة التجارية.
  - **Dashboard**: تصميم لوحة تفاعلية لعرض النتائج.

---

## 🚀 خطوات التنفيذ
1. **تنظيف البيانات**: التأكد من صحة التنسيقات ومعالجة القيم المفقودة.
2. **إضافة أعمدة جديدة**: مثل حساب الإيرادات لكل طلب.
3. **ربط الجداول**: دمج بيانات العملاء، المنتجات، والطلبات.
4. **الإجابة على الأسئلة التجارية**: مثل تحديد المدن الأعلى مبيعًا أو المنتجات الأكثر ربحية.
5. **إنشاء لوحة تحكم تفاعلية**: لعرض النتائج بشكل بصري يسهل فهمه.

---

## 📈 النتائج
- **رؤى العملاء**: تحديد المدن والمناسبات الأكثر مبيعًا.
- **الإيرادات**: مقارنة الأسعار مع المنافسين وتحديد المنتجات الأكثر ربحية.
- **التسليم**: تحليل أسباب التأخير وتأثيرها على رضا العملاء.
- **التخزين**: معالجة مشكلة التخزين الزائد في مواسم معينة مثل عيد الحب.
